# BOP Music
#### Local music player using MVVM design pattern

The application may serve as an example of
 - ViewModel
 - Room Database
 - Dagger-Hilt
 - Background Service
 - Motion Layout
 - MediaPlayer
 - MediaStore
 - RecyclerView
 - Media Controls
 - ViewPager2
 - Broadcasting intents
